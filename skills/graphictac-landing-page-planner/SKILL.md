# Graphictac Landing Page Planner

Use this skill when the task involves:
- landing page strategy
- page structure
- hero copy direction
- CTA strategy
- conversion path planning
- FAQ planning
- product education layout

## Goal

Create landing pages that help buyers understand fit quickly and move toward inquiry, sample, or purchase.

## Workflow

1. Define the page objective.
2. Identify the audience and traffic source.
3. Choose the primary CTA and secondary CTA.
4. Build the page in a logical decision-making order.
5. Keep the content modular so it can work on Shopify or other web platforms.

## Default Page Structure

1. Hero with clear fit and CTA
2. Key reasons to buy
3. Product or application detail
4. Compatibility or workflow context
5. Proof, examples, or comparisons
6. FAQ
7. Final CTA

## Deliverables

- page outline
- section-by-section message plan
- hero headline options
- CTA options
- objection-handling FAQ
- offer framing

## Guardrails

- Every section should help the buyer answer a decision question.
- Avoid decorative sections with no conversion purpose.
- Do not overstate compatibility or performance.
- Keep outputs platform-neutral unless the request is explicitly Shopify-specific.
